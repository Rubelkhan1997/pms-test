# Reports & Analytics Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build all operational and financial reports — Occupancy, Revenue, Reservation, Cashier, Housekeeping, Guest, and Night Audit — with Redis caching, PDF export, and Excel export.

**Architecture:** Reports are computed queries on the tenant DB. Large date ranges are queued via Horizon and the result URL returned when ready. Short queries (<30 days) are served synchronously with Redis cache (TTL varies by report type). PDF via DomPDF, Excel via Laravel Excel.

**Tech Stack:** Laravel 13, Redis, Laravel Horizon, DomPDF, Laravel Excel, Pest PHP.

**Depends on:** Phases 3, 5, 6 (data producing phases — reservations, folios, housekeeping).

---

## Tasks Overview

| Report | Key Metrics | Cache TTL |
|---|---|---|
| Occupancy | Arrivals, Departures, In-House, Occupancy %, RevPAR, ADR | 15 min |
| Revenue | Room rev, Extra charges, Payments, Outstanding, By segment | 15 min |
| Reservation | Bookings by source, Cancellations, No-Shows, Lead time | 30 min |
| Cashier | Collections by method, Refunds, Shift summaries | 5 min |
| Housekeeping | Rooms cleaned, Assigned, Pending, Avg clean time | 30 min |
| Guest | New, Repeat, VIP, Nationality breakdown | 1 hour |
| Night Audit | Business date history, Discrepancies | Permanent |
